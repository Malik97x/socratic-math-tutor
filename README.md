# Socratic Math Tutor (Research Edition)

A comprehensive, research-grade educational platform designed to guide students through complex math problems using Socratic inquiry.

## 🚀 Live Demo
**[Click here to visit the tutor](https://raheeq-lab.github.io/socratic-math-tutor/)**

---

## 🛠 Features
- **Socratic AI Assistant**: Powered by Groq (LLaMA 3), the assistant is strictly prompted to never give answers, only guiding questions.
- **5-Problem Curriculum**: Focuses on patterns, probability, and algebraic reasoning.
- **Digital Scratchpad**: Full-page A4 workspace with drawing and text support for tablet users.
- **Cloud Real-Time Sync**: Integrated with **Supabase** for centralized data collection of student answers, AI logs, and behavioral metrics.
- **Research Analytics**: Tracks "Frustration vs. Focus" time, pre-study confidence levels, and transition seconds.
- **Data Privacy**: Secure API key management via cloud configuration.

## 📊 Research Data Collection
All data is streamed directly to a central database. To export logs:
1. Double-click the header title on the dashboard to reveal the hidden **Data** tab.
2. Click **Export CSV** to get the full research dataset.
